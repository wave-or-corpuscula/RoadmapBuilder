### Реализовано

**Core:**
- GraphQL-подобный граф навыков с dependencies
- Learning Plan с ordered_skill_ids + skill_statuses
- Auth: register/login/refresh JWT
- API: plans, skills, users, progress

### Что нужно реализовать

1. **Структура steps в БД** ✅ ГОТОВО
   - JSONB поле `plan_steps` в learning_plans
   - Иерархия: skill → steps → substeps (до 3 уровней)
   - Step domain class с поддержкой вложенности
   - LearningPlan обновлён с методами steps
   - Репозиторий сохраняет/читает steps
   - Миграция 0004_add_plan_steps

2. **UI компоненты**
   - Панель навыка (по центру, расширенная)
   - Дерево шагов с визуальными маркерами статуса
   - Чат с ИИ внутри шага
   - Режим "Проверить знания" (AI задаёт вопросы)
   - Кнопка "Разбить на подшаги"

3. **AI интеграция**
   - Разбиение шага: промпт с контекстом графа, скилла, иерархии
   - Чат: RAG с документацией/ресурсами шага
   - Проверка знаний: интерактивный диалог с оценкой

### Ограничения
- Максимум 3 уровня вложенности подшагов
- AI разбивает только один шаг за раз
- **Шаг можно разбить только один раз** — повторное разбиение недоступно
