# 📍 ФАЗА 1 — Минимальная модель домена (Vertical Slice)

## 1.1 Выбор тестового домена

Берём:

- 20–30 skills    
- 1 категория (Execution & Core Syntax)
- Полные зависимости

Почему не весь junior → middle?

Потому что:

- Нам нужен тестовый полигон
- Не encyclopedic граф
- А инженерная проверка архитектуры

## 1.2 Формализация структуры Skill

```python

    id
    slug
    title
    description
    difficulty
```

```python

    from_skill
    to_skill
```
